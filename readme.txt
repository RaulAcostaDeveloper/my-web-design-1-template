Bienvenido a tu App
Las rutas de la app como /products se configuran a través de las carpetas en /app
El layout.tsx es el elemento que se va a renderizar a partir de la ruta actuál.
El layout.tsx es heredado por las rutas hijas.
El page.tsx es el elemento que se va a renderizar en la ruta actuál.
Las rutas se crean a través de las carpetas /app/producto.

Usa los styledComponents para construir tu web page.
Usa /pageConfig para construir tus paginas personalizadas.
Personaliza las propiedades de cada elemento.
Asegurate de usar entidades para caracteres especiales. Ejemplo: á: &aacute;
La cantidad ideal de elementos en un arreglo de información es la que viene por default.
